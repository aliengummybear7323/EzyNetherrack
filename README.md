# EzyNetherrack

A tiny Create add‑on that adds one simple but useful recipe:
**Haunt Gravel → Netherrack** using Create’s Haunting mechanic.

Perfect for early‑game Nether access, bulk processing, or automation setups.

---

## ✨ Features

- Adds a single, clean Create recipe  
- Haunt **Gravel** into **Netherrack**  
- Fully datapack‑style — no custom items or blocks  
- Lightweight, server‑friendly, and safe to add/remove  
- Works on any world, modpack, or server running Create

---

## 📦 Requirements

- **Minecraft:** 1.20.x  
- **Forge:** 47.4.x  
- **Create:** 0.5.1f (I know it is old, I will add support for a later version)

---

## 🛠 How It Works

Just run Gravel through any Create Haunting setup:

- Encased Fan + Soul Fire  
- Mechanical Haunting setups  
- Any modpack‑compatible haunting machine  

The output is **Netherrack**, at a 1:1 ratio.

---

## 🔧 Installation

1. Download the latest release from Modrinth  
2. Drop the `.jar` into your `mods/` folder  
3. Launch Minecraft with Create installed  
4. Enjoy your new automation path
